1. Название проекта
   Слияние двух чисел (Merging Two Numbers)

2. Описание
   Этот проект написан на Python.
   Программа берет два числа от пользователя и объединяет их в одно число.
   Например: если ввести 12 и 34, программа выдаст 1234.

3. Как работает программа
   Сначала пользователь вводит два числа.
   Потом программа превращает их в строки.
   После этого числа соединяются (склеиваются).
   И в конце выводится результат.

4. Пример работы

Введите первое число: 12
Введите второе число: 34
Результат: 1234

5. Код программы

a = input("Введите первое число: ")
b = input("Введите второе число: ")

result = a + b

print("Результат:", result)

6. Как запустить

1) Установить Python
2) Открыть файл с кодом
3) Запустить программу
4) Ввести два числа
